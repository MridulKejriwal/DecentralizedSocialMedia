# 🚀 Decentralized Social Media (Soroban Smart Contract)

## 📌 Project Description
This project is a decentralized social media smart contract built using Soroban on the Stellar blockchain. It enables users to create and store posts in a fully decentralized and censorship-resistant environment.

Unlike traditional social media platforms, this system does not rely on centralized servers, ensuring transparency, data ownership, and immutability.

## ⚙️ What It Does

- Allows users to create posts directly on-chain
- Stores user-generated content in a decentralized manner
- Enables retrieval of posts by user identity
- Removes dependency on centralized platforms

---

## ✨ Features

- 🧑‍💻 User-based posting system
- 📦 On-chain storage using Soroban
- 🔍 Fetch posts per user
- 🔐 Transparent and immutable data
- 🌐 Fully decentralized architecture
- ⚡ Built on Stellar's fast and low-cost network

---

## 🛠️ Tech Stack

- Soroban (Stellar Smart Contracts)
- Rust
- Stellar Blockchain

---

## 🚀 How to Use

1. Deploy the contract to Soroban testnet
2. Call `create_post(user, content)` to add a post
3. Call `get_posts(user)` to retrieve posts
